# const

用于限定修饰的値不可更改。 

### 修饰常规变量：

```c
//两者效果相同，a的値将不再可以修改
//所以声明的时候要初始化
int const a;
const int a;
```

define比较： 

define常量进行纯文本替换，可以用来声明数组的长度，而const只能修饰变量

### 修饰引用（既指针）

```c
//指向整形常量的指针，可以修改指针的值，但是不能修改指向的整形的值
//理解成 *pci 是常量
int const *pci
//指向整形的常量指针，无法修改指针的值，但是可以修改指向的整形的值
//理解成 pci 是常量
int *const cpi

int const * const cpci //都不能修改
```
