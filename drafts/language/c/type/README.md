# 基本数据类型

c中只有4种 -- 整型，浮点型，指针，聚合类型（数组和结构等）

K&R 允许隐式声明变量（只做了解，不要用）

没有初始化的变量，值是不可预料的；所有变量和地址的相关性，是编译器处理的。

### 布尔

c没有布尔类型

* 0是假，任何非零値都为真
* 所有空\(NUL，0, '\0' \)为条件判断的false，否则为true
