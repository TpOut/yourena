Java 使用 System.out 来表示标准输出设备， 而用 System.in 来表示标准输入设备 

Java 并不直接支持控制台输人， 但是可以使用 Scanner 类创建它的对象， 以读取来自 System.in 的输入

如下所示： Scanner input = new Scanner \(System.ln\); PrintWriter pw = new PrintWriter\(file\);
