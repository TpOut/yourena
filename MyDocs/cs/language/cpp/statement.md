

- 赋值语句

    产生的值是左操作数的值

- 表达式语句

    表达式执行会产生值

    ```C++
    var1 = (var2 = 3) + 2;
    ```

    - 逗号

        ```C++
        //逗号表达式也是一个结算点
        //即逗号之前的 表达式，一定比之后的表达式先执行
        //逗号表达式的值，是最后的表达式的值
        int c = (i= 20, j = 30, k = 35);
        cout << c; //此时c 是35
        ```

- 声明语句



`24*5 + 22*5`是先执行前者，还是后者，取决于系统实现
