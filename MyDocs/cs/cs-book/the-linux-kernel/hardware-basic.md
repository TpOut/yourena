cpu

- 高速缓存，主要存储器

buses

- 地址总线（数据地址），数据总线（数据读写），控制总线（时间和信号）

- ISA，PCI 

地址空间

- 外设所占用的内存空间，叫做I/O 空间。
- DMA（Direct Memory Access）
